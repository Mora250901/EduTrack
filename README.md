# EduTrack - Plataforma de Gestión del Bienestar Estudiantil

## ¿Qué es?
EduTrack es una plataforma web y móvil diseñada para ayudar a colegios de nivel inicial y primaria a detectar tempranamente señales de alerta en el aprendizaje y bienestar de sus estudiantes.

## Tecnologías utilizadas
- **Backend:** Node.js + Express
- **Base de datos:** PostgreSQL (Supabase)
- **Frontend Web:** React
- **Frontend Móvil:** React Native
- **ML Service:** Python + Flask + scikit-learn
